Nexus
    Secure Authentication Platform

    Identity &
    Access Control

    A full-stack MERN application with JWT authentication, bcrypt password hashing, and role-based access control for users and administrators.

    🔑

    JWT Authentication

    Stateless token-based authentication using JSON Web Tokens. Tokens expire in 24 hours and protect every sensitive API endpoint.

    🛡️

    Role-Based Access

    Three-tier authorization: public visitors, authenticated users, and administrators — each with distinct permissions and accessible routes.

    🔒

    Bcrypt Security

    All passwords are salted and hashed with bcrypt before storage. Plain-text passwords never touch the database.

    🗄️

    MongoDB + Mongoose

    User data is persisted in MongoDB with a structured Mongoose schema enforcing validation and data integrity.

    👥

    User Management

    Administrators can view all registered users, their roles, and remove accounts directly from the management dashboard.

    ⚡

    Express REST API

    Clean RESTful endpoints for registration, login, user listing, and deletion — all served from a single Node.js process.
